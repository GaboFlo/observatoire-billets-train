# Génération des icônes du site

Pour une compatibilité maximale avec tous les navigateurs et appareils, les fichiers d'icônes suivants doivent être générés à partir de `logo.png` :

## Fichiers requis

### Favicons standards
- `favicon.ico` (16x16, 32x32, 48x48) - Déjà présent
- `favicon-16x16.png` (16x16)
- `favicon-32x32.png` (32x32)
- `favicon-192x192.png` (192x192) - Pour Android
- `favicon-512x512.png` (512x512) - Pour Android

### Apple Touch Icons (iOS)
- `apple-touch-icon-57x57.png` (iPhone 3GS et antérieurs)
- `apple-touch-icon-60x60.png` (iPhone 4)
- `apple-touch-icon-72x72.png` (iPad)
- `apple-touch-icon-76x76.png` (iPad)
- `apple-touch-icon-114x114.png` (iPhone 4 Retina)
- `apple-touch-icon-120x120.png` (iPhone 5+)
- `apple-touch-icon-144x144.png` (iPad Retina)
- `apple-touch-icon-152x152.png` (iPad Retina)
- `apple-touch-icon-180x180.png` (iPhone 6+)

### Microsoft Tiles (Windows)
- `mstile-70x70.png` (70x70)
- `mstile-150x150.png` (150x150)
- `mstile-310x310.png` (310x310)

### Safari
- `safari-pinned-tab.svg` (SVG monochrome pour Safari)

## Outils recommandés

Vous pouvez utiliser des outils en ligne comme :
- https://realfavicongenerator.net/
- https://favicon.io/
- https://www.favicon-generator.org/

Ou des outils en ligne de commande comme ImageMagick pour générer toutes les tailles à partir de `logo.png`.

## Note

Tous ces fichiers doivent être placés dans le dossier `public/` pour être accessibles à la racine du site.
